[CSPro 23]收集卡牌·题解

题面

小林在玩一个抽卡游戏，其中有 n 种不同的卡牌，编号为 1 到 n 。每一次抽卡，她获得第 i 种卡牌的概率为 pi。如果这张卡牌之前已经获得过了，就会转化为一枚硬币。可以用 k 枚硬币交换一张没有获得过的卡。

小林会一直抽卡，直至集齐了所有种类的卡牌为止，求她的期望抽卡次数。

分析

抽卡过程是一个无记忆的离散时间马尔可夫过程：下一次抽卡对“是否集齐”的贡献仅由当前收集的卡牌情况和硬币数决定，无需考察更久远的历史。因此把卡牌情况 J 和 硬币数 c 作为状态的两个维度，定义随机变量 XS 表示从状态 S:=(J,c) 开始到集齐卡牌还需要的抽卡次数。根据全期望公式，对于下一次抽卡的 n 种可能，有：

E(XS)=∑i=1nE(XS|Bi)P(i)

其中 Bi 代表事件“从状态 S 出发，下一次抽中了编号为 i 的卡”。条件变量 XS|Bi 代表“从状态 S 出发的所有抽卡序列中，以 i 开头的序列的个数”，而抽完卡后的新状态 Si 对应的随机变量 XSi 和该变量之间的关系为（抽卡序列是独立同分布的）：

E(XS|Bi)=E(1+XSi)=1+E(XSi)

代入上式，得到：

E(XS)=∑i=1n[1+E(XSi)]P(i)=1+∑i=1nE(XSi)P(i)

而 XSi 可以根据卡牌 i 与当前卡牌集合 J 的关系进行分类：i∈J 和 i∉J。即：

XSi={X(J,c+1)if i∈J,X(J∪{i},c)otherwise.

于是得到了完整的状态转移方程。

实现

具体实现的时候，我们采用一个 n 位二进制数来表达集合 J 。由于抽卡序列是无穷的，正向递推难以确定显式边界条件；故考虑从 J0={1,1,1,…,1} 和 C=80 （想一想，为什么是 80 ？）开始逆向递推。边界条件为 ∀c,f(J0,c)=0 。

#include<bits/stdc++.h>
using namespace std;
const int N=1<<16,C=80;
double f[N][81];
int main(){
    int n,k;
    cin>>n>>k;
    vector<double> p(n);
    for(int i=0;i<n;i++)
        cin>>p[i];
    //f[j][c] ：从状态(j,c) 出发，集齐的期望抽卡次数
    int ff=(1<<n)-1;
    for(int j=ff;j>=0;--j){
        int cnt=__builtin_popcount(j); //已收集的卡牌张数
        for(int c=C;c>=0;--c){
            if(c>=k*(n-cnt)){
                f[j][c]=0.0;
                continue;
            }
            double res=1.0;
            for(int i=0;i<n;++i){
                if(j & (1<<i)) //已有
                    res += p[i] * f[j][min(c+1,C)];
                else
                    res += p[i] * f[j|(1<<i)][c];
            }
            f[j][c]=res;
        }
    }
    printf("%.10lf",f[0][0]);
    return 0;
}
