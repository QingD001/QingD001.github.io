A - Marisa Steals Reimu's Takeout

Description

The Darkness Brought In by Swallowstone Naturalis Historia

— Dateless Bar "Old Adam"

Marisa is a girl of integrity who always helps others safeguard their belongings. Over a period of n days, she comes each day to take one of Reimu's takeouts. The i-th takeout is described by its deliciousness value — an integer wi ( 0≤wi≤2), forming a sequence w of length n.

Marisa has a special fondness for the number 3. She can perform the following operation zero or more times:

  • Select a non-empty subsequence∗ of w whose sum is divisible by 3, and remove the elements of the subsequence from w.

Determine the maximum number of operations Marisa can perform.

∗A sequence a is a subsequence of a sequence b if a can be obtained from b by the deletion of several (possibly, zero or all) element from arbitrary positions.

solve

显然地贪心。由于 0≤wi≤2，0 肯定能单独贡献一次，再就是 (1,2) 或 (2,1) 。不一定能全部删完，剩下的 1 或者 2 尽量就行，它们能组成 3 的倍数显然都需要三个数就行。

xxxxxxxxxx
#include<bits/stdc++.h>
using namespace std;
int t;
int main(){
    cin>>t;
    while(t--){
        int n;cin>>n;
        int c[3]={0,0,0};
        for(int i=0,x;i<n;i++){
            cin>>x;++c[x];
        }
        cout<<c[0]+min(c[1],c[2])+abs(c[1]-c[2])/3<<endl;
    }
    return 0;
}

B. Remilia Plays Soku

Description

Remilia is trying to escape, while Reimu wants to land the final hit.

The playing field consists of n positions arranged in a circle. For each 1≤i<n, positions i and i+1 are adjacent, and positions 1 and n are also adjacent.

Initially, at time 0, Reimu is at position x1 and Remilia is at position x2, where x1≠x2.

Each second, the following happens in order:

  1. Remilia either moves to an adjacent position or stays in place. Over the entire game, she may move to an adjacent position at most k times.

  2. After observing Remilia's action, Reimu either moves to an adjacent position or stays in place.

  3. If they are at the same position after both actions, Reimu catches Remilia and the game ends.

Assuming both players play optimally, Reimu always tries to minimize the catching time, while Remilia tries to maximize it.

Find the number of seconds until Reimu catches Remilia.

Solve

Reimu 和 Remilia 一个期望尽量靠近，一个期望尽量远离。它们的初始距离为 min(abs(x1-x2),n-abs(x1-x2))，而 Remilia 最多能拖延 k 次。答案就是两者之和。

#include<bits/stdc++.h>

        if(n<=3){
            cout<<1<<"\n";
            continue;
        }
        cout<<k+min(abs(x1-x2),n-abs(x1-x2))<<"\n";
    }   
    return 0;
}

C. Cirno and Number (Easy/Hard Version)

Description

Led On by a Cow to Visit Zenkou Temple

— Neo-traditionalism of Japan

This is the hard version of the problem. The difference between the versions is that in this version, 1≤n≤10. You can hack only if you solved all versions of this problem.

You are given a non-negative integer a and a non-empty, strictly increasing sequence of digits d of length n, where 0≤di≤9.

Find the minimum value of |a−b| over all non-negative integers b whose decimal representation contains only digits from d.

形式化题意：给出了一个升序的集合 d，要用 d 中数字作为数位构造 b 使 |a−b| 最小，求这个最小值。

Solve

这是一个细节较多的数位贪心题。显然答案只可能来自两类候选：不超过 a 的最大合法数，以及不小于 a 的最小合法数，最后取二者与 a 的距离最小值。

对不超过 a 的候选，从高位到低位构造：若当前位可以与 a 对应位相等，则先保持相等；否则尝试填入小于当前位的最大合法数字，之后低位全部填最大合法数字。若当前位既不能相等也不能变小，则向前回退，找到一个可以降低的位置，将其降低后，后续位全部填最大合法数字。若同长度下无法构造，则枚举所有更短长度，取其中最大的合法数。

对不小于 a 的候选同理：若当前位可以相等则保持相等；否则尝试填入大于当前位的最小合法数字，之后低位全部填最小合法数字。若当前位既不能相等也不能变大，则向前回退，找到一个可以升高的位置，将其升高后，后续位全部填最小合法数字。若同长度下无法构造，则枚举所有更长长度，取其中最小的合法数。

实现时需注意首位不能填 0，但单独的数字 0 是合法的。由于 a≤1017，a 的长度最多为 18，因此枚举长度的代价可以忽略，整体复杂度为 O(t⋅18⋅n)。

D. Sanae, Cross and Color

Description

Faith Is for the Transient People

— Mountain of Faith

Upon the mountain, Sanae gazes at the stars. Faith, for her, is where winds encounter one another and choices diverge, and it is the point where all directions meet. For the glory of the Holy Cross, let us trace the sign of faith together.

There are n distinct integer points in the plane, where the i-th point is located at (xi,yi). To color the points, choose two integers k1 and k2 such that each of the four regions divided by the lines x=k1+0.5 and y=k2+0.5 contains at least one point. Each point (x,y) is colored according to the region it lies in:

  • Top-left ( x≤k1 and y>k2): the point is colored red.

  • Top-right ( x>k1 and y>k2): the point is colored green.

  • Bottom-left ( x≤k1 and y≤k2): the point is colored blue.

  • Bottom-right ( x>k1 and y≤k2): the point is colored yellow.

A valid coloring of the third test case, where k1=4 and k2=5.

Find the number of distinct colorings, where two colorings are considered distinct if and only if there exists at least one point colored differently, regardless of the choice of k1 and k2.

Solve

求不同染色的数量，n 的范围太大说明这不太可能是暴力问题，大概率存在计数规律。坐标可以先离散化，先用竖线从左往右扫，左边点集合 L 右边点集合 R；要求横线切分后四象限均非空，则横线纵坐标 p 应该满足 minL≤p≤maxL 且 minR≤p≤maxR 这样的 p 数量为：

max(0,min(maxL,maxR)−max(minL,minR))

由于坐标都是离散化后的，因此不同的 p 必然对应不同切分。最终复杂度为 O(nlog⁡n)

E1. Amanojaku and Sequence (Easy Version)

Description

Tinkerbell of Inequality

— Taboo Japan Disentanglement

This is the easy version of the problem. The difference between the versions is that in this version, q=1 and op=2. You can hack only if you solved all versions of this problem.

For a sequence s, let |s| denote its length.

For a non-negative integer sequence c, define f(c) as the sum of the squares of its prefix sums:

f(c)=∑i=1|c|(∑j=1icj)2.

Now define g(b,m) as follows.

Let b be an integer sequence such that bi≥−1 for every i, and let m be a non-negative integer. A non-negative integer sequence c is called valid for (b,m) if all of the following conditions hold:

  • |c|=|b|;

  • ∑i=1|c|ci=m;

  • for every 1≤i≤|b|, if bi≥0, then ci=bi; otherwise, if bi=−1, then ci may be any non-negative integer.

The value g(b,m) is defined as the sum of f(c) over all valid sequences c for (b,m). If there is no such sequence, then g(b,m)=0.

You are given an array a of length n, where ai≥−1, and q queries of the following one type:

  • Given three integers l, r, and m, compute g([al,al+1,…,ar],m) modulo 998244353, where [al,al+1,…,ar] denotes the subarray∗ of a from position l to position r.

∗An array a is a subarray of an array b if a can be obtained from b by the deletion of several (possibly, zero or all) elements from the beginning and several (possibly, zero or all) elements from the end.

形式化题意：定义 g(b,m) 为构造一个 c 使 c[i]=(b[i]==-1) ? any non-negative integer : b[i] 且 ∑c=m 的所有 c 对应的 f(c) 之和，现给出原始序列 a 与 m,l,r ，求 g(a[l→r],m) 。

Solve

Hard Version 还需要数据结构维护，根本不会，遂仅写本部分题解。本部分就是一个简单组合数学题。原序列中实际用到只有 a[l→r] ，重整下标为 a[1→r−l+1] 。对构造的 c 分成两部分考虑：1) ci=ai 的部分，设其总和为 s 2) 可以填充任意非负整值的部分，设总共有 k 个位置。

任意填充非负整数值即为解不定方程：

x1+x2+⋯+xk=T:=m−s(∗)

不必探究具体的 c 怎么构造，而是对每个位置考虑（在数学上即为变换求和号次序）。对于位置 i ，记录固定值前缀 Ai 表示 1) 部分中下标 ≤i 的和；不定值前缀 Yi 表示 2) 部分中下标 ≤i 的和。于是有：

g=∑i=1r−l+1∑∀c(Ai+Yi)2=∑i=1r−l+1(Ai2∑∀c1+2Ai∑∀cYi+∑∀cYi2)

由于 Ai 相关的量都是容易知道的，问题转化为求解：∑1,∑Yi,∑Yi2

由于方程 (∗) 的解共有 (T+k−1k−1) 种，这就是所有合法构造 c 的总数，即 ∑∀c1 的值。

另一方面：

∑Yi=∑x1+⋯+∑xhi

其中 hi 为小于等于 i 的最大可构造数的下标。由于所有的 c 对于每个 cij 都是等价平均参与分配的，所以可认为：

∑Yi=hi×T×(T+k−1k−1)k=hi×(T+k−1k)

再一方面类似前述的有：

∑Yi2=∑∑p=1hixp2+2∑∑1≤p≤q≤hixpxq

仍运用轮换对称性得到在 ∑ 意义下， ∑xp2⇔∑x12,∑xpxq⇔∑x1x2 。

而：

∑x12=∑x1+2∑(x12)

由组合意义知，∑(x12) 表示把所有 T 个球装进 k 个盒子（可以有空盒子）的所有方案中，从第一个盒子的 x1 个球里选出两个球的方法总数。这其实可以等价于，在 1 部分多了两个特殊分割板，因此 ∑(x12)=(T+k−1k+1)。故有：

∑x12=(T+k−1k)+2(T+k−1k+1)

而 ∑x1x2 的组合意义就是在所有分配方案中，再从第一个盒子选一个球、从第二个盒子选一个球的总方案数，同样可以理解为多了两个特殊分隔板，故同样地有 ∑x1x2=(T+k−1k+1)

最终得到：

∑Yi2=hi((T+k−1k)+2(T+k−1k+1))+2(h2)(T+k−1k+1)

所以：

g=∑i[Ai2(T+k−1k−1)+2Aihi(T+k−1k)+hi(T+k−1k)+hi(hi+1)(T+k−1k+1)]

因此，在代码中我们维护四个前缀统计量：

P1=∑iAi2,P2=∑iAihi,P3=∑ihi,P4=∑ihi(hi+1)

本题就轻松愉快的做完了。

ll qpow(ll a,ll b){
    ll res=1;
    while(b){
        if(b&1)res=res*a%MOD;
        a=a*a%MOD;
        b>>=1;
    }
    return res;
}
