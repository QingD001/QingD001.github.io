A Number Between Two Others

题目描述

给你两个整数 x 和 y ，使得 y>x 和 ymodx=0 （即 y 能被 x 整除）。

你的任务是确定是否存在一个整数 z，使得

  • z 位于 x 和 y 之间（即 x<z<y ）；

  • z 能被 x （即 zmodx=0 ）整除；

  • y 不能被 z 整除（即 ymodz≠0 ）。

思路分析

不妨设 y=kx,z=mx(k,m>1)，则要求 kmodm≠0 且 k,m∈N∗。我们要探讨 m 的存在性，显然存在一个构造是 m=k−1。若 m=k−1∈N∗，则存在；反之不存在。

Code

xxxxxxxxxx
#include<bits/stdc++.h>
using namespace std;
typedef long long LL;
int main(){
    int t;
    cin>>t;
    while(t--){
        LL x,y;
        cin>>x>>y;
        if(y/x==2)
            cout<<"NO\n";
        else
            cout<<"YES\n";
    }
    return 0;
}

Alternating String

题目描述

给出一个仅由 'a' 和 'b' 构成的字符串，你必须对该字符串选择一个子串进行翻转操作，同时可以选择是否反转其中字母（'a'变成'b'，'b'变成'a'）。问原字符串经过一步变换能否称为 'a' 和 'b' 交替出现的字符串？

思路分析

翻转+反转一定能把该子串塑造成想要的样子，只要看不匹配的段是否连续即可。

Code

xxxxxxxxxx
#include<bits/stdc++.h>
using namespace std;
bool check(string s, char start) {
    int n=s.size();
    vector<int> diff(n);
    for(int i=0;i<n;i++){
        char exp=(i % 2 == 0 ? start : (start == 'a' ? 'b' : 'a'));
        diff[i]=(s[i]!=exp);
    }
    int l=-1,r=-1;
    for(int i=0;i<n;i++){
        if (diff[i]){
            if(l==-1)l=i;
            r=i;
        }
    }
    if(l==-1)return true;
    for(int i=l;i<=r;i++) 
        if(!diff[i])return false;
    return true;
}
int main(){
    int t;cin>>t;
    while(t--){
        string s;cin>>s;
        if(check(s, 'a') || check(s, 'b'))
            cout << "YES\n";
        else
            cout << "NO\n";
    }
    return 0;
}

Red-bBlack Pairs

问题描述

有一张由 2×n 个单元格组成的表格。表格中的每个单元格都被涂成红色或黑色。您想重新绘制表格中的某些单元格，以便至少有一种方法可以将所有单元格分割成 $n$ 对，从而满足以下条件：

  • 每对单元格的颜色相同；

  • 每对单元格共享一条边。

至少有多少个单元格需要重新涂色？

思路分析

考虑前面 i−1 列都处理完后，第 i 列的处理方法。显然要么自己一列染同色，要么上下两格分别和 i+1 切去染色。预处理好两种转移的代价数组，这下就是个 dp 了！

Code

xxxxxxxxxx
#include<bits/stdc++.h>
using namespace std;
const int N=2e5+10,INF=0x3f3f3f3f;
int main(){
    int t;cin>>t;
    while(t--){
        int n;cin>>n;
        string s[2];
        cin>>s[0]>>s[1];
        vector<int> cv(n+1),ch(n+1); 
        for(int i=1;i<=n;i++){
            if(s[0][i-1]==s[1][i-1])cv[i]=0;
            else 
                cv[i]=1;
            if(i<n)
                ch[i]=(s[0][i-1]!=s[0][i])+(s[1][i-1]!=s[1][i]);
        }
        vector<int> dp0(n+1,INF),dp1(n+1,INF);
        dp0[0]=0;
        for(int i=1;i<=n;i++){
            if(dp0[i-1]!=INF){
                dp0[i]=min(dp0[i],dp0[i-1]+cv[i]);
                if(i<n)
                    dp1[i]=min(dp1[i],dp0[i-1]+ch[i]);  
            }
            if(dp1[i-1]!=INF)
                dp0[i]=min(dp0[i],dp1[i-1]);
        }
        cout<<dp0[n]<<endl;
    }
    return 0;
}

Exceptional Segments

题目描述

给你两个整数 n 和 x。

考虑序列 [1,2,3,…,n] 。您需要找出包含 x 且 XOR 等于 0 的子序列的个数。换句话说，你需要计算 (l,r) 中包含 1≤l≤x≤r≤n 和 l⊕(l+1)⊕⋯⊕r=0 的数对 (l,r) 的个数。

例如，如果 n=7 和 x=6 ，那么以下线段是合适的：

  • (4,7) ，因为 x 位于该分段中， 4⊕5⊕6⊕7=0也位于该分段中；

  • (1,7) ，因为 x 位于该分段和 1⊕2⊕3⊕4⊕5⊕6⊕7=0 中。

由于答案可能非常大，因此输出它的模数 998244353 。

思路分析

和本题强相关地，有一个经典的 trick：

f(n)=(⊕k=1nk)={nnmod4=01nmod4=1n+1nmod4=20nmod4=3

根据 xor 的自反性，即求有多少数对 (l,r) 使 1≤l≤x≤r≤n 且 f(r)=f(l−1)

由单调性，显然只需要统计 f(l−1)=f(r)=0/1 的匹配数。

Code

xxxxxxxxxx
#include<bits/stdc++.h>
using namespace std;
typedef long long ll;
const ll mod=998244353;
ll f(ll L,ll R,int k){
    if(L>R)return 0;
    if(L<=k)return (R>=k?(R-k)/4+1:0);
    return (R-k)/4-((L-1-k)/4);
}
int main(){
    int t;cin>>t;
    while(t--){
        ll n,x;cin>>n>>x;
        ll L0=f(1,x-1,3)+1,R0=f(x,n,3);
        ll L1=f(0,x-1,1),R1=f(x,n,1);
        ll ans=((L0%mod)*(R0%mod)%mod+(L1%mod)*(R1%mod)%mod)%mod;
        cout<<ans<<"\n";
    }
    return 0;
}
